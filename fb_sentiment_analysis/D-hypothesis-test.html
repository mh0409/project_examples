MY472 Final Exam Part D: Testing a Substantive Hypothesis

To conclude this assignment, you will offer preliminary evidence regarding one potential explanation about why there is so much negativity on Facebook comments: negative comments are widespread because they receive more engagement. In other words, maybe negative comments generated the type of reactions on people that make them more likely to like those comments or to reply to those comments.

library("dplyr")

# Open SQLite connection
fbdb <- dbConnect(RSQLite::SQLite(), "fbdata.sqlite")
  1. Do negative comments receive more likes than neutral or positive comments? Use any statistical or graphical methods that you consider appropriate to answer this question.
  • Negative comments do receive more comments than neutral or positive (i.e. non-negative) comments.
  • I performed a two-sided hypothesis test where the null hypothesis (that the difference between the means of likes received by negative and non-negative comments) is 0 and found that the p-value indicated that we could reject the null and that there is a statistically significant difference between the means.
  • Having done so, I then performed a one-sided hypothesis test where the null hypothesis was that the mean likes received for negative comments was higher. The resulting p-value indicated that we could not reject the null hypothesis.
# Find likes_count for comments group by comments.sent_name
summary_cmt_likes <- dbGetQuery(fbdb, 
           "SELECT sent_name, AVG(likes_count) as 'avg_likes'
           FROM comments
           GROUP BY sent_name")

# Get all comments and respective sentiment and like counts
df_cmt_likes <- dbGetQuery(fbdb, 
           "SELECT id, sent_name, likes_count
           FROM comments")

# Rename positive and neutral to non-negative for hypothesis testing
df_cmt_likes[df_cmt_likes$sent_name == "positive", 2] <- "nonnegative"
df_cmt_likes[df_cmt_likes$sent_name == "neutral", 2] <- "nonnegative"
# Perform 2-sided hypothesis test (null case -- the treatment has no effect)
cmt_likes_2test <- t.test(likes_count ~ sent_name, data = df_cmt_likes)
cmt_likes_2test
## 
##  Welch Two Sample t-test
## 
## data:  likes_count by sent_name
## t = 38.351, df = 8308931, p-value < 2.2e-16
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  0.6502220 0.7202625
## sample estimates:
##    mean in group negative mean in group nonnegative 
##                  2.010215                  1.324973
# Perform 1-sided hypothesis test (null = "greater" )
cmt_likes_1test <- t.test(likes_count ~ sent_name, mu = 0, alternative = "greater", data = df_cmt_likes)
